OPML feed scraper and data aggregation tool for Pinchflat backup workflows — built with
TypeScript, Playwright for dynamic content extraction, and a custom Node.js proxy server
for CORS handling. Iteratively developed with focus on practical utility: scrapes OPML
feed sources, normalizes metadata, and produces structured output for downstream archival workflows.
